# Prompt Garden

Prompt Garden 是一个发现、导入和收藏优质提示词的平台。通过导入码，你可以快速将 Garden 中的提示词引入 Prompt Optimizer 使用或收藏。

## 导入码格式

导入码是一个字符串标识符，有两种形式：

### 基础格式

```
NB-001
```

直接使用导入码，系统会使用默认示例参数。

### 带示例选择

```
NB-001@ex-2
```

在导入码后追加 `@` 和示例 ID，系统会使用指定示例的参数填充变量。

## 如何使用导入码

### 步骤

1. 在 Prompt Garden 网站找到想要的提示词
2. 复制其导入码
3. 在 Prompt Optimizer 中点击顶部工具栏的 **导入** 按钮
4. 粘贴导入码
5. 选择目标工作区（可选）
6. 确认导入

### 导入选项

导入时有两种模式：

**直接使用**

- 提示词写入当前工作区
- 变量自动填充默认值或示例参数
- 适合立即测试和使用

**保存到收藏**

- 提示词、变量、媒体资源一起保存为收藏
- 来源信息自动记录（导入码、来源链接）
- 适合建立个人提示词库

## 导入后会发生什么

### 提示词处理

- 文本提示词写入对应工作区
- 变量自动识别并填充默认值
- 如果选择了示例，使用示例参数填充

### 工作区匹配

系统会根据提示词类型自动选择合适的工作区：

| 提示词类型 | 目标工作区 |
|-----------|-----------|
| 系统提示词 | 系统提示词工作区 |
| 用户提示词 | 用户提示词工作区 |
| 多消息上下文 | 多消息工作区 |
| 变量化模板 | 变量工作区 |
| 文生图提示词 | 文生图工作区 |
| 图生图提示词 | 图生图工作区 |

### 收藏保存

如果选择保存到收藏：

- 提示词内容完整保存
- 变量定义和默认值保留
- 图片和媒体资源一并保存
- 记录导入码和来源链接
- 支持后续从收藏恢复使用

## 常见问题

### 导入码无效怎么办

- 检查导入码是否复制完整
- 确认导入码对应的内容是否仍然有效
- 尝试刷新页面后重新导入

### 如何查看导入历史

导入的内容会进入对应工作区的历史记录。如果保存到收藏，可以在收藏管理器中查看。

### 收藏后如何更新

如果导入码对应的内容有更新，重新导入并保存到收藏会自动更新已有收藏（基于导入码去重）。

### 与直接导入 JSON 的区别

- **导入码**：从 Prompt Garden 拉取最新内容，包含来源信息
- **JSON 导入**：导入本地文件，不包含来源信息
